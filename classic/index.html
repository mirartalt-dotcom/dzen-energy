Нейронка для голосового ввода

Голос превращается в текст прямо в телефоне. А чтобы текст сам раскладывался по ползункам, нужна нейронка — вставь ключ одного из сервисов. Без ключа работает простой разбор по словам.

Напоминание в Telegram

У нас есть бот @dzen_energy_bot. Жми кнопку, в боте нажми «Старт» — и каждый вечер в выбранное время будет приходить напоминание замерить день. Семь серий — потом бот сам поздравит с финалом сезона.

  1. Открой бота кнопкой ниже.
  2. Нажми «Старт» (или «Restart»).
  3. Готово. Отключить — командой /stop в любой момент.
